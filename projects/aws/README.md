# AWS configurations
In this section, I will describe the AWS Terraform configurations that I'm learning. 

## Prerequisites
1. [Terraform CLI Installation](../../README.md)
2. [AWS CLI for mac](https://docs.aws.amazon.com/cli/latest/userguide/install-cliv2-mac.html)
3. [Create an AWS Account](https://aws.amazon.com/free/?all-free-tier.sort-by=item.additionalFields.SortRank&all-free-tier.sort-order=asc&awsf.Free%20Tier%20Types=*all&awsf.Free%20Tier%20Categories=*all)
4. AWS credentials
   * Create `Access Key` and `Secret Key` in AWS console
   * Add this information using `aws config`
     ```shell
    AWS Access Key ID [None]: 
    AWS Secret Access Key [None]: 
    Default region name [None]: us-west-1
    Default output format [None]:
     ```
    > You can check your credential data in `~/.aws/credential`
